## Critical Distinction
Claude described this as AI "controlling" signals. The vendor's own documentation says it "recommends." NCDOT's spokesperson specifically says "maintaining oversight by engineering staff." Three sources converge on the same point: this is a monitoring and recommendation tool, not an autonomous one. Claude collapsed this distinction entirely.

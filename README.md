# AI Reading Exercises — PLAN A6613

**Course:** PLAN A6613: AI and the Future of Cities (Spring 2026)  
**Author:** Daniel Hardesty Lewis  
**Institution:** Columbia GSAPP

## Overview

Weekly AI Reading Exercises for Prof. Wittels' course. Each assignment uses a generative AI tool to research a specific urban technology question, conducts human-in-the-loop verification, and synthesizes findings into a one-page academic document.

## Repository Structure

| Week | Topic | AI Tool |
|------|-------|---------|
| [Week 3](week3/) | The Economic Impact of Urban Technologies | Claude 3.5 Sonnet |
| [Week 4](week4/) | Place-Based Innovation Policy | Claude 3.5 Sonnet |
| [Week 5](week5/) | From Digital to Physical Infrastructure | Claude Opus 4.6 (Antigravity IDE) |

Each `weekN/` directory contains:
- LaTeX source (`.tex`) and compiled PDF
- `*.tex.d/refs/` — Downloaded and annotated source materials
- `prompt.md` — Prompt strategy and iteration log
